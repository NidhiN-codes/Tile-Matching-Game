# Tile Matching Game

A simple interactive tile matching memory game built using HTML, CSS, and JavaScript.

## Live Demo
Play the game here:  
https://nidhin-codes.github.io/Tile-Matching-Game/

## Features
- Interactive tile flipping mechanism
- Memory-based matching logic
- Simple and clean UI
- Smooth gameplay using vanilla JavaScript

## Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation)

## What I Learned
- Handling user interactions in JavaScript
- Working with arrays and game logic
- DOM manipulation and event handling
- Building interactive UI without frameworks

## How to Run Locally
1. Clone this repository
2. Open `index.html` in your browser
3. Start playing!
